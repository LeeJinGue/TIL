## Command
- git flow init: 현재 완료된 add, commit 등 git의 상태를 확인하는 명령어
- git flow feature start <feature-name>: 해당 feature를 추가하는 branch를 따온다.
- git flow feature finish <feature-name>: 해당 feature의 추가를 마치고 본래의 branch로 돌아갑니다.
